# MCBEMoreDropItems
 Minecraft Windows 基岩版客户端 百倍掉落物（`1.21.51.01+`）


### 使用方法
>  
- 将编译后的dll文件注入到 Minecraft.Windows.exe 程序  
- 支持卸载


### 使用库
 - [lighthook](https://github.com/SamuelTulach/LightHook)
 - [HookManager](https://github.com/cngege/HookManager)
